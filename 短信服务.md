# 需求：在现有登录页面集成阿里云短信验证码功能

## 一、阿里云配置信息

配置项通过环境变量设置，请参考 `server/.env.example`：

```
ALIYUN_SMS_ACCESS_KEY_ID=你的AccessKeyId
ALIYUN_SMS_ACCESS_KEY_SECRET=你的AccessKeySecret
ALIYUN_SMS_ENDPOINT=dypnsapi.aliyuncs.com
ALIYUN_SMS_SIGN_NAME=你的签名名称
ALIYUN_SMS_TEMPLATE_CODE=你的模板CODE
```

## 二、需要使用的 API

使用阿里云 **号码认证服务**（不是短信服务），SDK 依赖：`@alicloud/dypnsapi20170525`

只涉及两个接口：

### 1. 发送验证码 — `SendSmsVerifyCode`

### 2. 校验验证码 — `CheckSmsVerifyCode`

## 三、已实现的功能

- 手机号格式校验：`^1[3-9]\d{9}$`
- 发送频率限制：60秒内不可重复发送
- 验证码校验：通过 `verifyResult === 'PASS'` 判断
- 延迟初始化：未配置密钥时不会崩溃

## 四、注意事项

- 这是 **号码认证服务**，不是「短信服务」
- 验证码的生成、存储、有效期管理全部由阿里云服务端处理
- `CodeType` 在使用 `##code##` 时是**必填**参数
